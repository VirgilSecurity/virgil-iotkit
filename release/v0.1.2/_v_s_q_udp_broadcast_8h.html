Detailed Description

UDP broadcast network interface implementation.

VSQUdpBroadcast is based on VSQNetifBase class. It implements SNAP protocol based on UDP broadcast packets. It can be used for any network supporting UDP broadcast messages.

You can add VSQUdpBroadcast class as one of implementations for VSQIoTKitFacade. Also do not forget to add VSQFeatures::SNAP_INFO_CLIENT feature :

auto features = VSQFeatures() << VSQFeatures::SNAP_INFO_CLIENT;
auto impl = VSQImplementations() << QSharedPointer<VSQUdpBroadcast>::create();
if (!VSQIoTKitFacade::instance().init(features, impl, appConfig)) {
VS_LOG_CRITICAL("Unable to initialize Virgil IoT KIT");
return -1;
}
